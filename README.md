# pdfReview

## 介绍
记录PDF文件常用的修订方式，主要介绍PC端Acrobat Reader、Foxit福昕、金山PDF、okular等电脑软件，iOS移动端Goodnotes、Notability等手写批注软件，LaTeX修订宏包changes、todonotes等LaTeX源码修订方式。

Happy LaTeXing！~
